# Introduction to C Programming

## What is C?

C is a **general-purpose procedural programming language** developed by **Dennis Ritchie** in **1972** at **Bell Laboratories**.

It is one of the most popular and powerful programming languages used for:

* System Programming
* Operating Systems
* Embedded Systems
* Game Development
* Compiler Design

---

# Features of C Language

* Simple and Easy to Learn
* Fast Execution Speed
* Portable Language
* Structured Programming
* Rich Library Functions
* Supports Low-Level Programming
* Efficient Memory Management

---

# Structure of a C Program

```c id="92v1ol"
#include <stdio.h>

int main()
{
    printf("Hello World");

    return 0;
}
```

---

# Explanation of Program

## `#include <stdio.h>`

* Header file
* Used for input and output functions

---

## `int main()`

* Main function of the program
* Execution starts from here

---

## `printf()`

* Used to display output on screen

Example:

```c id="1d9dm0"
printf("Hello World");
```

---

## `return 0;`

* Indicates successful execution of program

---

# Basic Rules in C

| Rule              | Description               |
| ----------------- | ------------------------- |
| Semicolon `;`     | Ends statements           |
| Case Sensitive    | `A` and `a` are different |
| Main Function     | Mandatory                 |
| Curly Braces `{}` | Define blocks             |

# Keywords in C

Keywords are reserved words with predefined meaning.

Examples:

* `int`
* `float`
* `char`
* `if`
* `else`
* `return`

---

# Identifiers in C

Identifiers are names given to:

* Variables
* Functions
* Arrays

Example:

```c id="cywe5p"
int age;
```

Rules:

* Cannot start with number
* Cannot use spaces
* Cannot use keywords
